Создайте небольшой бэкенд для библиотеки с двумя моделями:

Модели:

Author

id — уникальный идентификатор автора

name — имя автора

books — список книг, которые написал автор (many-to-many)

Book

id — уникальный идентификатор книги

title — название книги

authors — список авторов книги (many-to-many)

Один автор может написать несколько книг, а одна книга может принадлежать нескольким авторам. 
Реализуйте добавление новых авторов с книгами, получение списка всех авторов с их книгами, 
добавление книги к уже существующему автору и получение списка всех книг с их авторами.
